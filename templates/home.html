Common questions

Do I need a DPR to use Gradmate?

No. You can manually mark courses as completed or in-progress from the Course Catalog. The DPR upload is just the fastest way to get started — it fills everything in automatically.

Does it work for my catalog year?

Gradmate supports multiple catalog years. When you set up your profile, you pick your program and catalog year, and the requirement engine uses the matching rules for that year automatically.

Which programs are supported?

Currently supports B.S. Computer Science and B.S. Information Technology. M.S. Computer Science and MBA profiles can be created, with requirement data added by an admin.

Is Gradmate free?

Yes, completely free. Gradmate is a senior capstone project built to help CSUN students, not a commercial product.
